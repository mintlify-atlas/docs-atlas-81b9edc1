  <Steps>
    <Step title="Install the package">
      Choose your platform and install WAX using your package manager.

      <Tabs>
        <Tab title="TypeScript">
          ```bash npm
          npm install @hiveio/wax
          ```
          ```bash yarn
          yarn add @hiveio/wax
          ```
          ```bash pnpm
          pnpm add @hiveio/wax
          ```
        </Tab>
        <Tab title="Python">
          ```bash
          pip install hiveio-wax
          ```
        </Tab>
      </Tabs>
    </Step>

    <Step title="Initialize WAX">
      Create a WAX instance for offline operations or connect to the Hive network.

      <Tabs>
        <Tab title="TypeScript">
          ```typescript
          import { createWaxFoundation, createHiveChain } from '@hiveio/wax';

          // For offline operations (transaction building, signing)
          const wax = await createWaxFoundation();

          // For online operations (API calls, broadcasting)
          const chain = await createHiveChain();
          ```
        </Tab>
        <Tab title="Python">
          ```python
          from wax import create_wax_foundation, create_hive_chain

          # For offline operations (transaction building, signing)
          wax = create_wax_foundation()

          # For online operations (API calls, broadcasting)
          chain = create_hive_chain()
          ```
        </Tab>
      </Tabs>
    </Step>

    <Step title="Build and sign a transaction">
      Create a transaction, add operations, and sign it with your preferred signing provider.

      <Tabs>
        <Tab title="TypeScript">
          ```typescript
          // Create a transaction
          const tx = await wax.createTransaction();

          // Add operations
          tx.pushOperation({
            vote: {
              voter: "alice",
              author: "bob",
              permlink: "example-post",
              weight: 10000
            }
          });

          // Sign and broadcast
          await tx.sign(wallet, publicKey);
          await chain.broadcast(tx.transaction);
          ```
        </Tab>
        <Tab title="Python">
          ```python
          # Create a transaction
          tx = wax.create_transaction()

          # Add operations
          tx.push_operation({
              "vote": {
                  "voter": "alice",
                  "author": "bob",
                  "permlink": "example-post",
                  "weight": 10000
              }
          })

          # Sign and broadcast
          await tx.sign(wallet, public_key)
          await chain.broadcast(tx.transaction)
          ```
        </Tab>
      </Tabs>
    </Step>
  </Steps>
