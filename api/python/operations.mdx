title: Operations
description: Hive blockchain operation types for transactions
---

Operations represent actions on the Hive blockchain. Each operation is a protocol buffer message that can be added to transactions.

## Importing operations

All operations are available from `wax.proto.operations`:

```python
from wax.proto.operations import (
    transfer,
    vote,
    comment,
    transfer_to_vesting,
    withdraw_vesting,
    account_update2,
    # ... and many more
)
```

## Operation types

The `Operation` type is a union type alias:

```python
from wax.models.operations import Operation, WaxMetaOperation, ProtoOperation

# Operation can be:
# - Proto operation (protobuf Message)
# - Protocol operation (dict or str in HF26 format)
operation: Operation = transfer(...)

# WaxMetaOperation is used internally (Message or OperationBase)
wax_operation: WaxMetaOperation = transfer(...)
```

<ResponseField name="Operation" type="TypeAlias">
  Union of `ProtoOperation | ProtocolOperation` - any operation format.
</ResponseField>

<ResponseField name="WaxMetaOperation" type="TypeAlias">
  Union of `Message | OperationBase` - internal operation representation.
</ResponseField>

<ResponseField name="ProtoOperation" type="TypeAlias">
  Union of `dict[str, Any] | str | Message` - types convertible to proto.
</ResponseField>

<ResponseField name="ProtocolOperation" type="TypeAlias">
  Union of `dict[str, Any] | str` - HF26 protocol format.
</ResponseField>

## Transfer operations

### transfer

Transfers HIVE or HBD between accounts.

```python
from wax import create_wax_foundation
from wax.proto.operations import transfer

wax = create_wax_foundation()

op = transfer(
    from_account="alice",
    to_account="bob",
    amount=wax.hive.satoshis(1000),  # 1.000 HIVE
    memo="Payment for services"
)
```

<ParamField path="from_account" type="str" required>
  Sender account name.
</ParamField>

<ParamField path="to_account" type="str" required>
  Receiver account name.
</ParamField>

<ParamField path="amount" type="NaiAsset" required>
  Amount to transfer (HIVE or HBD in NAI format).
</ParamField>

<ParamField path="memo" type="str" default="">
  Transfer memo (can be encrypted).
</ParamField>

### recurrent_transfer

Sets up recurring transfers between accounts.

```python
from wax.proto.operations import recurrent_transfer

op = recurrent_transfer(
    from_account="alice",
    to_account="bob",
    amount=wax.hive.satoshis(100),
    memo="Monthly payment",
    recurrence=24,  # Every 24 hours
    executions=12   # 12 times total
)
```

<ParamField path="from_account" type="str" required>
  Sender account name.
</ParamField>

<ParamField path="to_account" type="str" required>
  Receiver account name.
</ParamField>

<ParamField path="amount" type="NaiAsset" required>
  Amount per transfer.
</ParamField>

<ParamField path="memo" type="str" default="">
  Transfer memo.
</ParamField>

<ParamField path="recurrence" type="int" required>
  Hours between transfers (minimum 24).
</ParamField>

<ParamField path="executions" type="int" required>
  Total number of transfers to execute.
</ParamField>

## Voting operations

### vote

Votes on a post or comment.

```python
from wax.proto.operations import vote

op = vote(
    voter="alice",
    author="bob",
    permlink="my-awesome-post",
    weight=10000  # 100% upvote
)

# Downvote
op = vote(
    voter="alice",
    author="bob",
    permlink="spam-post",
    weight=-10000  # 100% downvote
)

# Remove vote
op = vote(
    voter="alice",
    author="bob",
    permlink="my-post",
    weight=0
)
```

<ParamField path="voter" type="str" required>
  Voting account name.
</ParamField>

<ParamField path="author" type="str" required>
  Post author account name.
</ParamField>

<ParamField path="permlink" type="str" required>
  Post permlink.
</ParamField>

<ParamField path="weight" type="int" required>
  Vote weight from -10000 (100% downvote) to 10000 (100% upvote). Use 0 to remove vote.
</ParamField>

## Content operations

### comment

Creates a post or comment.

```python
from wax.proto.operations import comment

# Create a post
post = comment(
    parent_author="",
    parent_permlink="hive",  # Category/tag
    author="alice",
    permlink="my-first-post",
    title="Hello Hive!",
    body="This is my first post on Hive.",
    json_metadata='{"tags":["hive","introduction"],"app":"my-app/1.0"}'
)

# Create a comment
comment_op = comment(
    parent_author="bob",
    parent_permlink="bobs-post",
    author="alice",
    permlink="re-bobs-post-20240101t120000",
    title="",
    body="Great post!",
    json_metadata="{}"
)
```

<ParamField path="parent_author" type="str" required>
  Parent post author (empty string for top-level posts).
</ParamField>

<ParamField path="parent_permlink" type="str" required>
  Parent post permlink (category for top-level posts).
</ParamField>

<ParamField path="author" type="str" required>
  Author account name.
</ParamField>

<ParamField path="permlink" type="str" required>
  Post/comment permlink (must be unique for the author).
</ParamField>

<ParamField path="title" type="str" required>
  Post title (empty for comments).
</ParamField>

<ParamField path="body" type="str" required>
  Post/comment body content.
</ParamField>

<ParamField path="json_metadata" type="str" required>
  JSON metadata string (tags, app info, etc.).
</ParamField>

## Power operations

### transfer_to_vesting

Powers up HIVE to Hive Power.

```python
from wax.proto.operations import transfer_to_vesting

op = transfer_to_vesting(
    from_account="alice",
    to_account="alice",
    amount=wax.hive.satoshis(100000)  # 100 HIVE
)

# Power up to another account
op = transfer_to_vesting(
    from_account="alice",
    to_account="bob",
    amount=wax.hive.satoshis(50000)
)
```

---

### withdraw_vesting

Starts power down process (13 weeks).

```python
from wax.proto.operations import withdraw_vesting

# Start power down
op = withdraw_vesting(
    account="alice",
    vesting_shares=wax.vests.satoshis(1000000000)  # Amount to power down
)

# Stop power down
op = withdraw_vesting(
    account="alice",
    vesting_shares=wax.vests.satoshis(0)
)
```

---

    delegatee="bob",

## Custom operations

### custom_json

Executes custom JSON operations (for apps and protocols).

```python
from wax.proto.operations import custom_json
import json

# Follow operation
op = custom_json(
    required_auths=[],
    required_posting_auths=["alice"],
    id="follow",
    json=json.dumps([
        "follow",
        {
            "follower": "alice",
            "following": "bob",
            "what": ["blog"]
        }
    ])
)

# Reblog operation
op = custom_json(
    required_auths=[],
    required_posting_auths=["alice"],
    id="reblog",
    json=json.dumps([
        "reblog",
        {
            "account": "alice",
            "author": "bob",
            "permlink": "great-post"
        }
    ])
)
```

<ParamField path="required_auths" type="list[str]" default="[]">
  Accounts requiring active authority.
</ParamField>

<ParamField path="required_posting_auths" type="list[str]" default="[]">
  Accounts requiring posting authority.
</ParamField>

<ParamField path="id" type="str" required>
  Custom operation ID (e.g., "follow", "reblog", "community").
</ParamField>

<ParamField path="json" type="str" required>
  JSON string containing operation data.
</ParamField>

## Complete example

```python
import asyncio
from wax import create_hive_chain
from wax.proto.operations import (
    transfer,
    vote,
    comment,
    custom_json
)
import json

async def main():
    chain = create_hive_chain()
    
    try:
        tx = await chain.create_transaction()
        
        # Create a post
        tx.push_operation(
            comment(
                parent_author="",
                parent_permlink="hive",
                author="alice",
                permlink="my-post",
                title="Hello Hive",
                body="My first post using Wax!",
                json_metadata=json.dumps({
                    "tags": ["hive", "wax"],
                    "app": "my-app/1.0"
                })
            )
        )
        
        # Vote on it
        tx.push_operation(
            vote(
                voter="alice",
                author="alice",
                permlink="my-post",
                weight=10000
            )
        )
        
        # Send a tip
        tx.push_operation(
            transfer(
                from_account="alice",
                to_account="bob",
                amount=chain.hive.satoshis(1000),
                memo="Thanks for the help!"
            )
        )
        
        # Follow someone
        tx.push_operation(
            custom_json(
                required_auths=[],
                required_posting_auths=["alice"],
                id="follow",
                json=json.dumps([
                    "follow",
                    {
                        "follower": "alice",
                        "following": "bob",
                        "what": ["blog"]
                    }
                ])
            )
        )
        
        # Sign and broadcast
        # ... (signing code) ...
        
    finally:
        chain.teardown()

asyncio.run(main())
```
